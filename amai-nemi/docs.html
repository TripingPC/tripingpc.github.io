Documentation for Amai and Nemi

Introduction

Amai and Nemi are bots for Discord. They were made as two different bots so that server owners can easily only get useful commands and/or the whacky stuff.

In light of this. This document will be color coded accordingly. Commands written in orange are for Amai, and commands written in green are for Nemi. And commands for both are going to be in default black and white colors.

Markov chatbots

Both Amai and Nemi have built-in learning markov-chain based chatbots. To use any of them, type amai, <whatever> or nemi, <whatever>. I recommend you to not type in a whole ton of stupid stuff. as they learn EVERYTHING. that also means, Dont put personal information you dont want others to see.
