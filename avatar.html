Reset My Avatar

A tool to revert your VRChat avatar to the default robot for Non-VRChat accounts

If you are using a VRChat.com account, just go to the avatars page and click on "Reset to Default Avatar".

In most cases, if you're logging in with a Steam account, you should be able to click the link and get back in right away.



RESET AVATAR NOW!


If you use an Oculus Quest:

You can still reset the avatar on your Oculus account if someone sent you a Quest crasher. But you will need to install the Oculus Rift version of VRChat and launch it in Desktop Mode to reset your avatar.

Download the Oculus Software installer and run it. The install is around 5 gigabytes in size and may take up to an hour.

Follow the setup instructions and connect with the same credentials as on the iOS/Android Oculus app when you set your Quest up for the first time.

Search for "VRChat" in the store and install it.

Launch the game once then close it. You will see a window called "VRChat Installation Helper". Click on "Proceed" and accept the administrator prompt.

Now you should be able to use the RESET AVATAR NOW! button to launch into the ResetMyAvatar world.




If the link doesn't work:

If you're playing on PC and you never accepted the prompt for setting up world launch links. You can find the setup application in VRChat's installation folder. It will be called install.exe

For Steam, right-click the game in your library, under Manage click Browse local files.


For Oculus, in Library, click the three dots next to the game title for VRChat, then click on Details.

Then click on the location to copy it to your clipboard.

Finally, open the Windows Run dialog by pressing the Windows key and R, and paste the folder from the clipboard, and click Run. It will open the folder.




It didn't change my avatar!?

Don't panic, that can happen if the avatar is very small and therefore loads too fast for the world to change it. there are ways to combat this.

Clearing cache:

You can follow the "alternative" instructions from the VRChat documentation to manually delete the cache. if you already have the avatar downloaded and it's cuasing it to load too quickly.


Slowing down your internet:

If deleting the cache wasn't enough, go in and delete it again. And right before launching the game. go to Speedtest.net and start a speed test, which will eat as much internet bandwidth on your PC as possible, making any downloads in VRChat much slower. Giving the tool some time to switch your avatar.




If none of it worked:

Welp, I'm sorry for you... I suppose you can request an avatar reset by sending in a ticket to the VRChat Team via this link and hope for the best. :(
